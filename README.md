# nhentai-vertical-scroll

Loads all nHentai images onto a single page for vertical scrolling.

*Note: If some images don't load, try refreshing.*

Only available for Firefox (contributors welcome!)

**Downloads**
- Firefox Marketplace: https://addons.mozilla.org/en-US/firefox/addon/nhentai-vertical-scroll/
- Local ZIP: https://github.com/the-final-coomer/nhentai-vertical-scroll/releases/latest
